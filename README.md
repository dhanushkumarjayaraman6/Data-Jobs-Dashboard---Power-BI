# Data Jobs Dashboard - Power BI Project

## Overview

This project is an interactive **Power BI dashboard** developed to
analyze global data job trends, salaries, job roles, seniority levels,
employee types, and geographical distribution of opportunities.

The report provides multiple pages with different visualization
techniques to help users explore insights related to the data jobs
market.

## Dashboard Pages

### 1. Home Page

-   Navigation menu to access all report sections.
-   User-friendly dashboard layout.

### 2. Data Jobs Dashboard

Includes: - Salary KPI cards - Performance rating analysis - Department
distribution - Job title distribution - Promotion history insights -
Year-wise job count analysis - Company-wise job role treemap

### 3. Data Jobs Analytics

Includes: - Annual and monthly salary summaries - Salary star rating -
Salary comparison by job title - Permanent employee analysis - Detailed
analytics table

### 4. Column & Bar Charts

Includes: - Annual salary by job title - Monthly salary by country -
City-wise salary comparison - Permanent employee analysis using stacked
charts

### 5. Link & Area Charts

Includes: - Day salary trends - Applicant funnel analysis - Country-wise
salary comparison - Job title relationships using combination charts

### 6. Common Charts

Includes: - Pie charts - Donut charts - Treemap visualizations - Scatter
plots - Seniority level analysis

### 7. Map Charts

Includes: - Filled maps by country - Bubble maps by city - Salary
distribution across geographical regions - Employee distribution
worldwide

### 8. Uncommon Charts
Includes:
- Advanced and less frequently used visualizations
- Specialized charts for unique analytical requirements
- Comparative analysis using innovative chart types

### 9. Tables
Includes: - Detailed tabular representation of the dataset- Job title-wise salary
information- Country and city-wise records - Interactive sorting and filtering
capabilities

### 10. Cards
Includes: - KPI cards displaying important metrics- Annual salary 
summary- Monthly salary summary- Employee count indicators- 
Performance rating highlights

### 11. Slicers
Includes:
- Dynamic filtering options
- Country-based filtering
- Job title filtering
- Seniority level filtering
- Permanent employee status filtering
- Enhanced user interaction across all report pages

## Key Metrics Analyzed
